# Dashboard-de-H-bitos-Tracking-Personal
Una mini aplicación web creada con React + TypeScript para gestionar notas rápidas tipo “post-it”. Permite crear, editar y eliminar notas, con persistencia automática usando localStorage. Proyecto simple, liviano y útil para practicar conceptos de frontend moderno.

Aplicación web simple de notas rápidas (tipo Post-it), desarrollada con React + TypeScript.  
Permite crear, editar y eliminar notas de forma instantánea, con persistencia automática mediante localStorage.

Funcionalidades:
- Crear nuevas notas.
- Editar texto en tiempo real.
- Eliminar notas.
- Guardado automático usando `localStorage`.
- Diseño responsive con grilla adaptable.

Tecnologías:
- React
- TypeScript

Instalación:
```bash
clonar
cd 
npm install
npm run dev
